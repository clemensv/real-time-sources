# Support

This project is maintained as a community effort. It is **not** a
Microsoft-supported product, and there is no SLA.

## Where to get help

| Question type                         | Best place                                    |
|---------------------------------------|-----------------------------------------------|
| Bug report                            | [GitHub Issues](https://github.com/clemensv/real-time-sources/issues) |
| Feature request / new source          | [GitHub Issues](https://github.com/clemensv/real-time-sources/issues) |
| How-to / usage question               | [GitHub Discussions](https://github.com/clemensv/real-time-sources/discussions) |
| Security vulnerability                | See [SECURITY.md](SECURITY.md) — do not file a public issue |
| Issue with the **upstream data source** | Contact the upstream API operator directly |

When filing an issue, please include:

- The source / feeder name (e.g. `bafu-hydro`)
- Whether you're running the container image, the Fabric notebook, or
  a local Python checkout
- Reproduction steps with the exact command and (sanitized) env vars
- Relevant log output

## Response expectations

This is a side project. Maintainers respond on a best-effort basis,
typically within a week. Pull requests with a clear, minimal fix and
green CI are merged fastest.
