## Summary

Sensor.Community is the standout for open, zero-auth environmental monitoring at scale. PurpleAir has superior freshness and density in North America but requires an API key. openSenseMap is a flexible platform with academic backing. Safecast provides unique radiation data under CC0. CWOP/APRS-IS is fascinating as a real-time citizen weather stream over legacy ham radio infrastructure.
