Microsoft Fabric · Real-Time Analytics

Real-time data sources — ready to stream into Fabric

Pre-built containers. Pull. Run. Analyze. 24 bridges streaming live data from weather stations, vessel trackers, energy markets, and more.

Browse Sources How It Works

How It Works

1

Pull the container

Each bridge is a pre-built Docker image on GitHub Container Registry.

docker pull ghcr.io/clemensv/real-time-sources/noaa
→
2

Stream to Event Hub

Point the container at your Event Hub, Fabric Event Stream, or any Kafka endpoint.

docker run -e KAFKA_BOOTSTRAP_SERVERS=... -e KAFKA_TOPIC=... ghcr.io/clemensv/real-time-sources/noaa
→
3

Query in real time

Data lands in your Fabric Eventhouse. Query with KQL instantly.

WaterLevelObservations | where timestamp > ago(1h) | summarize avg(value) by station
