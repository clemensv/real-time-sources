Connect to Fabric

1

Run Locally with Docker

Start a bridge on your workstation or jump box and point it directly at Event Hubs or Fabric Event Streams.

docker run \ -e KAFKA_BOOTSTRAP_SERVERS=<host>:9093 \ -e SASL_PASSWORD='<conn-string>' \ ghcr.io/...
2

Deploy on Azure Container Instances

Use the pre-built ARM templates or Azure CLI for lightweight managed deployment near your streaming stack.

az container create \ --resource-group myRG \ --image ghcr.io/... --environment-variables ...
3

Ingest into Fabric Event Streams

Create a custom endpoint in Event Streams, paste the Kafka settings, then route into Eventhouse tables for KQL.

StormAlerts | where timestamp > ago(1h) | summarize count() by alert_type
