⟐ Real-Time Sources
GitHub Database Docs

24 pre-built container bridges streaming live data from public APIs into Apache Kafka, Azure Event Hubs, and Fabric Event Streams. Click a row for deployment commands.

Getting Started

  1. Create an Event Hub
    Create an Azure Event Hub namespace or a Fabric Event Stream custom endpoint.
  2. Copy the connection string
    Get the SASL_PASSWORD from your Event Hub's Shared Access Policy.
  3. Pick a source
    Choose one or more bridges from the table. Click the row for deployment commands.
  4. Run the container
    Use docker run, az container create, or the ARM deploy button.

Docker Run

docker run \ -e KAFKA_BOOTSTRAP_SERVERS=<host>:9093 \ -e KAFKA_TOPIC=noaa \ -e SASL_USERNAME='$ConnectionString' \ -e SASL_PASSWORD='<conn-string>' \ ghcr.io/clemensv/real-time-sources/noaa

KQL — Create Target Table

.create table NOAATides ( timestamp: datetime, station_id: string, value: real, quality: string )
