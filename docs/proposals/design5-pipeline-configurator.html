How to Connect to Fabric

1

Create an Event Stream endpoint

Provision a custom endpoint in Fabric Event Streams or an Azure Event Hub namespace with Kafka enabled.

Event Stream ├── Custom endpoint └── Kafka bootstrap + SASL/PLAIN
2

Select the bridges you need

Choose one feed or several. The configurator builds a multi-container package and bundles KQL for all selected sources.

Selected: - NOAA NDBC - DWD - Bluesky - USGS Earthquakes
3

Deploy and query in Eventhouse

Run locally, deploy to ACI, or translate the compose file into your existing orchestrator. Then route into Eventhouse tables and query with KQL.

StormAlerts | where timestamp > ago(1h) | summarize count() by source
