# ENTSO-E Transparency Platform Containers (Kafka, MQTT/UNS, AMQP 1.0)

ENTSO-E market, load, generation, and cross-border flow data supports grid operations, trading analytics, public transparency, and energy-transition research. This source ships three independent containers that share one acquisition core and publish the same CloudEvents contract to Kafka, MQTT 5, or AMQP 1.0.

## Images

```bash
docker pull ghcr.io/clemensv/real-time-sources-entsoe-kafka:latest
docker pull ghcr.io/clemensv/real-time-sources-entsoe-mqtt:latest
docker pull ghcr.io/clemensv/real-time-sources-entsoe-amqp:latest
```

## Kafka variant

```bash
docker run --rm -e ENTSOE_SECURITY_TOKEN='<token>' -e CONNECTION_STRING='<connection-string>' ghcr.io/clemensv/real-time-sources-entsoe-kafka:latest
```

| Variable | Description | Default |
|---|---|---|
| `ENTSOE_SECURITY_TOKEN` | ENTSO-E API token. | required |
| `CONNECTION_STRING` | Event Hubs/Fabric Event Stream Kafka-compatible connection string. | — |
| `KAFKA_BOOTSTRAP_SERVERS` | Kafka bootstrap servers if not using `CONNECTION_STRING`. | — |
| `KAFKA_TOPIC` | Kafka topic. | — |
| `SASL_USERNAME`, `SASL_PASSWORD` | SASL PLAIN credentials. | — |
| `KAFKA_ENABLE_TLS` | Enable TLS. | `true` |

## MQTT/UNS variant

```bash
docker run --rm -e ENTSOE_SECURITY_TOKEN='<token>' -e MQTT_BROKER_URL='mqtt://broker:1883' ghcr.io/clemensv/real-time-sources-entsoe-mqtt:latest
```

| Variable | Description | Default |
|---|---|---|
| `MQTT_BROKER_URL` | MQTT broker URL `mqtt://host:1883` or `mqtts://host:8883`. | required |
| `MQTT_AUTH_MODE` | `anonymous`, `password`/`userpass`, or `entra`. | `anonymous` |
| `MQTT_USERNAME`, `MQTT_PASSWORD` | Username/password for brokers that require them. | — |
| `MQTT_TLS` / `MQTT_ENABLE_TLS` | Enable TLS. | false unless `entra` |
| `MQTT_CLIENT_ID` | MQTT client id. | generated by broker/client |
| `MQTT_ENTRA_CLIENT_ID` | User-assigned managed identity client id for Event Grid namespace MQTT. | — |
| `MQTT_ENTRA_AUDIENCE` | Entra audience. | `https://eventgrid.azure.net/` |

Topics are rooted at `energy/eu/entsoe/transparency/...`, with branches for `by-domain`, `by-psr-type`, and `cross-border`. Messages are QoS 1, binary-mode CloudEvents, and not retained because the feeder emits point events from rolling API windows.

## AMQP 1.0 variant

```bash
docker run --rm -e ENTSOE_SECURITY_TOKEN='<token>' -e AMQP_BROKER_URL='amqp://user:pass@broker:5672/entsoe' ghcr.io/clemensv/real-time-sources-entsoe-amqp:latest
```

For Azure Service Bus with Entra ID:

```bash
docker run --rm -e ENTSOE_SECURITY_TOKEN='<token>' -e AMQP_HOST='<namespace>.servicebus.windows.net' -e AMQP_ADDRESS='entsoe' -e AMQP_AUTH_MODE='entra' -e AMQP_ENTRA_CLIENT_ID='<uami-client-id>' -e AMQP_TLS='true' ghcr.io/clemensv/real-time-sources-entsoe-amqp:latest
```

| Variable | Description | Default |
|---|---|---|
| `AMQP_BROKER_URL` | Convenience URL `amqp(s)://user:pass@host:port/address`. | — |
| `AMQP_HOST`, `AMQP_PORT`, `AMQP_ADDRESS` | AMQP broker host/port/address. | `localhost`, 5672/5671, `entsoe` |
| `AMQP_AUTH_MODE` | `password`, `entra`, or `sas`. | `password` |
| `AMQP_USERNAME`, `AMQP_PASSWORD` | SASL PLAIN credentials. | — |
| `AMQP_TLS` | Enable TLS. Required for Azure. | false unless `entra` |
| `AMQP_ENTRA_AUDIENCE`, `AMQP_ENTRA_CLIENT_ID` | Entra CBS settings. | Service Bus scope, default credential |
| `AMQP_SAS_KEY_NAME`, `AMQP_SAS_KEY` | SAS-token CBS settings for SAS-only targets/emulators. | — |

## Common variables

| Variable | Description | Default |
|---|---|---|
| `ENTSOE_DOMAINS` | Comma-separated EIC bidding-zone codes. | DE-AT-LU, FR, NL, ES, DE |
| `ENTSOE_DOCUMENT_TYPES` | Comma-separated document type codes. | A75,A44,A65,A69,A70,A71,A72,A73,A74,A68,A11 |
| `ENTSOE_CROSS_BORDER_PAIRS` | Comma-separated `in>out` EIC pairs for A11. | major EU interconnections |
| `POLLING_INTERVAL` | Poll cycle seconds. | 900 |
| `ENTSOE_LOOKBACK_HOURS` | Initial lookback window. | 24 |
| `STATE_FILE` | Persistent checkpoint path. | `/mnt/fileshare/entsoe_state.json` in containers |
| `ONCE_MODE` | Run one cycle and exit. | false |
